API documentation

Base URL: —

Authentication: none required. Your generated key is for identifying your own projects — it isn't checked or enforced by any endpoint here.

Endpoints:

  • POST /compress-image — form field file, query param quality (1–95)
  • POST /convert-document-to-pdf — form field file (.docx, .pptx, or .txt)
  • POST /convert-image-to-pdf — form field file
  • POST /watermark-image — form field file, query params text, mode (single/tiled), position, opacity, angle
  • POST /generate-qr-code — query params text, size, error_correction (L/M/Q/H)

Sample request:

curl -X POST ".../compress-image?quality=50" -F "file=@photo.jpg" -o compressed.jpg

Max file size: 5MB per request. Requests are rate-limited to 10 per minute per IP (5/minute for key generation).
